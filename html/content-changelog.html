Highlighted topics - Angular

Check out all highlighted topics

Publish date: 9/19/2025

Angular & AI integration

  • Building AI-Powered features: How to integrate AI into Angular apps using APIs like Gemini and Firebase AI Logic, and exploring advanced, on-device AI with Gemma models, WebNN and WebLLM.
  • Generative UI & Agentic Programming: Concepts and architectural patterns for building applications where the UI is dynamically generated by Large Language Models (LLMs) and where AI plays a central role in the user experience.
  • AI-Enhanced Development Workflows: Using AI to improve the development process, including code generation, bug fixing, refactoring, and automated test writing, especially with the Angular CLI's Model Context Protocol (MCP).

Core Angular features & reactivity

  • Signals & fine-grained reactivity: This is a major area of focus.
    • Migration: How to transition from RxJS to Signals for state management, while also knowing when to continue using RxJS.
    • Advanced APIs: Deep dives into linkedSignal, computed, effect, afterRenderEffect and the new resource API.
    • Forms: A dedicated topic on Signal-based Forms, including the new experimental forms API and best practices.
    • Performance: Discussing the benefits of zoneless change detection and how it impacts performance and debugging.
  • Server-Side Rendering (SSR) & Hydration:
    • Incremental Hydration: Why it's a "game-changer" for loading times.
    • Streaming & Deferred Rendering: The future of SSR with concepts like streamed SSR and @defer blocks.
  • Component & architecture:
    • Headless UI: Creating reusable components without a pre-defined UI framework.
    • Animations: Using new built-in APIs for enter and leave animations.

Tooling, testing & workflows

  • Testing:
    • Component testing: Using modern tools like Playwright (aka Testronaut) and Vitest.
    • E2E testing: Generating end-to-end tests with tools like an MCP Playwright server.
    • Forms: A dedicated topic on Signal-based Forms, including the new experimental forms API and best practices.
    • Advanced techniques: Testing with the Angular TestBed and component harnesses, as well as unit testing RxJS and Signals.
  • Dev workflows: Improving the developer experience with topics like AI-assisted Test-Driven Development (TDD) and "Charted Coding" structured workflows.\

Architectural & advanced topics

  • Micro frontends: Implementing this pattern in an Angular context, often with Module Federation.
  • State management: A comprehensive comparison of different solutions: Angular Signals and community-based solutions.
  • Interoperability: How Angular works seamlessly with third-party libraries like TanStack and Formish.
